# 💻GalaxyTours  website

Este é uma aplicação em Front-end desenvolvida para a GalaxyTours uma empresa que precisava de um sistema funcional para maior beleza da página,fazer agendamento de turismo,responsibilidade além de subrir a falta de interatividade com para com esta agência de turismo.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)

## 🚀 Funcionalidades Implementadas
* **Responsibilidade Dinâmica (Tablets):**
Adaptação do layout para telas de tablets, garantindo estea organizado (texto e estruturas), com usabilidade e sem quebra.

* **Responsibilidade Dinâmica (Mobile):**
Otimização completa da interface para o Mobile, acesso mesmo em telas reduzidas.
Garantindo que estja organizado (texto e estruturas), com usabilidade e sem quebra.

* **Limpar a página Mobile:** Limpa a dica que só tem para desktop em tempo real fazendo com que a página esteja limpa.
