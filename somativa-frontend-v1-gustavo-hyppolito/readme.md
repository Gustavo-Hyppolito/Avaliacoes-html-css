# 💻 BellaPizza website

Este é uma aplicação em Front-end desenvolvida para a BellaPizza uma empresa que precisava de um sistema funcional para maior agilidade,fazer pedidos, organização interna da empresa além de subrir a falta de interatividade com a internet para com esta pizzaria.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)

## 🚀 Funcionalidades Implementadas
* **Saudação Dinâmica:** Mensagem de boas-vindas que muda conforme o horário local do usuário.
* **Cálculo em Tempo Real:** Formulário que processa e exibe resultados assim que o usuário digita exemplo: qunatiade a se pagar no rodízio.
* **Injeção de Conteúdo:** Adição de novos sabor para montar sua pizza na tela utilizando manipulação de DOM.
* **Cancelamento de ordem:** Limpa seus pedidos em tempo real fazendo com que possa recomeçar se necessario.
